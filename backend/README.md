# VibeWorker Backend

## 文件结构说明

- `app.py` - FastAPI 入口文件
- `config.py` - 配置管理
- `prompt_builder.py` - System Prompt 拼接引擎
- `sessions_manager.py` - 会话管理
- `tools/` - 5 个内置工具
- `graph/` - Agent 编排 (LangGraph)
- `workspace/` - System Prompt 文件
- `memory/` - 记忆存储
- `sessions/` - 会话 JSON 文件
- `skills/` - Agent Skills
- `knowledge/` - RAG 知识库文档
- `storage/` - 索引持久化存储
