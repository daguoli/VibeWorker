# 获取最近上映电影

## 描述
此技能通过访问豆瓣电影等平台获取当前正在热映的电影列表。

## 使用方法
### 步骤 1: 访问电影页面
使用 `fetch_url` 工具访问豆瓣电影热映榜单：`https://movie.douban.com/chart` 或 `https://movie.douban.com/cinema/nowplaying/`。

### 步骤 2: 解析数据
从返回的内容中提取电影标题、评分、导演及上映日期。

### 步骤 3: 汇总展示
将信息整理为清晰的列表返回给用户。

### 备注
- 豆瓣电影页面结构可能发生变化，需灵活解析。
- 如果直接访问受限，可尝试搜索相关新闻。
